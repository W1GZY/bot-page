Why SeanBot?

SeanBot is a highly specialized toolkit designed for communities that need real reliability and power.

Whether you are running a multi-server network, scheduling cross-timezone events, or protecting your server from raids, SeanBot gives you granular control without clutter. Tailor every module and keep your community running flawlessly.

Featured Modules

Web Dashboard & Core

Generate secure, single-use authentication links to manage your server settings directly from the browser, and sync your command layouts.

Server Moderation & Transcripts

Full suite of moderation tools including kick, ban, timeout, purgetimeout, and file-based chat history transcript archiving.

Staff Tools & Appeals

Manage ban appeals with review channels, configure preset warning reasons, and manage community suggestion boards with automatic voting.

Member Reports

Give users a clean UI to report rule-breakers with custom context menus, routing details directly to staff audit logs.

Audit Logging & Quarantine

Asynchronous multi-webhook audit logger with batch processing, rate-limit protection, and non-destructive member quarantine jail.

Leave of Absence (LOA)

Streamline administrative staff leaves with natural date parsing, auto-approvals, expiration warnings, and automated role synchronization.

Anti-Raid Protection

Keep your server safe automatically. Detects sudden bursts of spam, quarantines bad actors, blocks phishing links, and lets you test security settings with Canary Mode.

Advanced Raid Polls

Organize raid lobbies, team-ups, and events with interactive rosters, automatic waitlisting, slot caps, and roster exports.

Raid & Event Organizer

Schedule server raids with interactive reaction slots, custom strategy notes & image modal popups, and O(1) live status embeds.

Team-Up Federation Mesh

Broadcast live Team-Up lobbies across allied servers in your network with synced proxy embeds and automatic cross-server thread creation.

Community Calendar

Auto-updating interactive event calendar synced with Google Sheets, featuring developer notes and visual categorization.

Auto-Pin System

Empower your community to pin great content! Messages achieving reaction thresholds get pinned automatically with custom channel overrides.

Activity Tracker

Track member participation across text, voice, and commands with automated Active/Inactive role assignments and rate-limited background processing.

Embed Builder

Interactive in-chat workspace to draft, design, and save rich embeds. Bind templates to automated triggers like welcome messages or custom commands.

Emoji Suite

Enterprise emoji management. Track usage stats, restrict custom emojis to specific roles, prevent reaction spam, and replace text aliases via webhooks.

Reaction Suite & Analytics

Analyze reaction trends, react with custom animated emojis, sweep bot reactions, and ping reactor groups with single-click shortcuts.

Auto Mention Replies

Configure personal automated replies triggered when other members mention you in chat, equipped with profanity filtering and cooldowns.

Returning Member Roles

Automatically re-assign designated roles to returning members who leave and rejoin the server within a 2-year window.

Beastmoon Tracker

Track Wizard101 Beastmoon form progress, calculate Lunari and Idol costs, view stats, and compete on federated network leaderboards.

Wizard101 Pet Tome

Fuzzy-search and autocomplete 1,000+ pets directly from Google Sheets to inspect hatching flags, wow factors, and acquisition sources.

Roshambo Battle Royale

Interactive elemental and spirit school battle game featuring multi-player lobbies, custom elimination rules, and sudden death rounds.

W101 Item Search

Instantly query the Wizard101 gear database to inspect item schools, required levels, rarities, set bonuses, and Archmastery stats.

Level Scaling Calculator

Calculate exact stat downscaling when playing at lower level tiers with dynamic database cap lookups and school selections.

Server Administration & Utilities

Manage roles, channels, slowmodes, toggle modular plugins, and query member avatars, role details, and server statistics.

Custom Server Prefixes

Configure custom text command prefixes per server with instant database updates and global fallback handling.

System & RAM Diagnostics

Monitor hardware memory usage, process loads (Bot, Worker, Listener, MongoDB), and system cache with permission-gated metrics.

Fun & Entertainment

Generate fake Discord quotes, browse cat breeds, fetch random cat facts, render custom cat image macros, and ask the magic 8-ball.

Server Federation

Connect multiple Discord servers together! Share live channels, sync forum threads, and maintain a shared ban list across your network.

Event Reminders & Timezones

Schedule automatic alerts mentioning roles at the right time. Save your local timezone so every reminder is perfectly synced globally.

Personal Reminders

Right-click any message to set a quick reminder. SeanBot will DM you when time is up, with snooze options directly from chat.

Dynamic Voice Channels

Join a hub channel to instantly create your own private Voice room with full kick, ban, and lock controls. Auto-deletes when empty.

Server Logging & User Reports

Keep clean logs of server activity. Groups rapid events to prevent spam, archives deleted messages, and provides a reporting UI for members.

Wizard101 Toolkit

Calculate multi-buff blade/trap/weakness damage outputs and export compact deck share codes into visual images.

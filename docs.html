Quick Start Guide

Welcome to SeanBot! Follow these three simple steps to integrate SeanBot into your Discord community:

1. Invite the Bot

Add SeanBot to your server with basic administrative permissions. Ensure the bot role is positioned above roles it needs to manage in your Server Settings.

2. Manage Active Modules

SeanBot features a modular architecture. Disable tools you don't need using /serveradmin module or via the Web Dashboard to keep your server clutter-free.

💡 Pro Tip: Use /staff diagnose anytime to verify bot permissions and command setup across your channels.

3. Set Command Prefix & Language

While slash commands are enabled by default, you can configure text command prefixes using /prefix set [prefix].

Anti-Raid Protection

Protect your server from sudden bot raids, link spam bursts, and suspicious mass joins automatically.

Key Features

  • Heat Score System: Dynamically calculates threat levels based on message frequency, user age, and repetitive links.
  • Canary Mode (Safe Testing): Test security settings without punishing actual members. Logs actions to staff channels first.
  • Quarantine Isolation: Temporarily revokes roles from suspected raid accounts and moves them to a secure holding channel.

Primary Commands

  • /antiraid dashboard — View real-time threat scores and server health.
  • /antiraid canary [on/off] — Enable or disable simulation mode.
  • /antiraid whitelist [role] — Grant trusted roles immunity from threat detection.

Server Federation & Relays

Cross-server communication networks made simple. Sync channels, forum threads, and banlists between multiple Discord servers.

Setting Up a Network

  1. Run /thread network create [name] on your primary hub server to receive a network key token.
  2. On secondary servers, execute /thread network join [token].
  3. Link specific channels together with /relay link [relay_id].
🔒 Security: All cross-server relays use HMAC token verification to guarantee message integrity.

TimePing & Global Reminders

Eliminate timezone confusion for global gaming and community events.

Usage Guide

  • Personal Timezone: Save your local timezone with /remindme or right-click any message to attach a reminder.
  • Scheduled Pings: Use /timeping schedule to configure automated alerts that ping specific roles at interval count-downs (e.g., 24h, 1h, 15m prior).
  • Native Event Sync: Automatically attach TimePings to official Discord Scheduled Events using /timeping native.

Event Signups & Team-Ups

Organize MMO raids, dungeon runs, and competitive lobbies with interactive signup embeds.

Key Commands

  • /teamup create — Open an interactive modal form to host a dungeon team-up.
  • /raid start — Create a multi-role raid signup board with slot limits per role (e.g., Tank, Healer, DPS).

When group requirements are met, SeanBot can automatically create a private temporary voice channel for the participants.

Wizard101 Toolkit

High-tier utilities built specifically for Wizard101 communities.

Features

  • Live Pet Tome: Search over 1,200+ pets directly via Google Sheets sync (or use the web Pet Tome page).
  • Damage Assistant: Calculate multi-buff blade, trap, and weakness damage outputs using /damage.
  • Deck Sharing & Image Export: Render compact deck codes into high-resolution images with /deck lookup and /decks export.
  • Beastmoon Tracker: Store and showcase your Beastmoon form tiers with /bm forms and /bm leaderboard.

Staff Tools & Ban Appeals

Empower your moderation staff with custom reason presets, appeal processing, and suggestion boards.

Ban Appeals Setup

Enable direct-message ban appeals via /appeal setup. Banned users receive a submission link, which routes their appeal to your staff review channel for voting.

Frequently Asked Questions

Why aren't slash commands showing up?

Make sure SeanBot was granted the applications.commands OAuth2 scope when invited. You can re-invite the bot using the invite link on the homepage.

How often does the Pet Tome database update?

The database synchronizes automatically every 1 hour directly from the collaborative Google Sheet.
